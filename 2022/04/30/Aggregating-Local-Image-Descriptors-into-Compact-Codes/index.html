Aggregating Local Image Descriptors into Compact Codes

Abstract

This paper addresses the problem of large-scale image search.
We first present and evalute different ways of agregating local image descripers into a vector and show that the Fisher kernel achieves better performance than the reference bag-of-visual words approach for any given vector dimension.
We then jointly optimize dimensionality reduction and indexing in order to obtain a precise vector comparision as well as preserving high accuracy.

1. Introduction

Image search consisists here of finding in a large set of images the ones that represent the same object or location possibly viewed from different viewpoints and in the presence of occlusions and clutter.

Typical applications include finding locations and particular objects, as well as detecting partial image duplicates on the web and deformed copies.

The bag-of-words representation is the most widely adopted method for this purpose.
There are three reasons for the success of the BOW representation.
First, this representation benefits from powerful local descriptors, such as the SIFT descriptor[7] or more recent ones[8],[9],[10].
Second, the BOW vector representation can be compared with standard distance.
Third, the BOW vector can be high dimensional, in which case the vector are sparse and inverted lists can be used to implement efficient search.
However, two factors limit the number of images that can be indexed in practice: the search time itself, which becomes prohibitive when considering more than 10 million images, and the memory useage per image.

In this paper, we address the problem of searching for the most similar images to given query in a very large image database (a hundred million images).

Most similar to our work is the approach of jegou et al., which proposes an approximate nearest neighbor search for BOW features. However, an image needs to be encoded by more than a hundred bytes to approximate the search accuracy of the inital representaion.

In contrast, our approach obtains a higher accuracy with a very compact representation. This performance is achieved by optimizing:

  1. the representaion, how to aggregate local image descriptors into a vector representation,
  2. the dimensionality reduction of these vector,
  3. the indexing algorithm.

A low-dimesional vector can be indexed efficiently, but its discriminative power may be too low.

Our first contirbution is an image representaion of moderate dimensionality that provides an excellent search accuracy.
Our representation is based on Fisher kernels[20], a power tool to transform an incoming variable-size set of independent samples into a fixed-size vector representation.
We assume that the samples are distributed according to a parametric generative model, in this case a Gaussian Mixture Model estimated on a trainin set.
Experimental results demonstrate that this choice significantly outperforms BOW for the same size on moset data set.
Moreover, it is cheaper to compute because fewer visual words are required. We also evaluate a simplified versionof Fishers kernels, the Vector of Local Aggregated Descriptors (VLAD).

The description vectors are compared using the L2 distance.
Their dismensionality is reduced to a few hundred components by principal component analysis (PAC).

The accoracy is measured by mAP.
